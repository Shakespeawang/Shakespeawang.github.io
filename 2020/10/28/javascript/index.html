网页从外部载入javascript文件方法：

是一种脚本语言，用于编写网页页面，通常放在html的body或者head中，也可同时存在两部分中；

对于脚本语言，是不需要编译的，在读到哪一行就执行哪一行，因此如果后面有错误，前面的也能执行

变量的作用域：

局部变量：在函数中通过var声明的变量； 全局变量：在函数外script中声明的变量；

注意：没有用var声明就使用的变量，就默认都是全局变量，不论这个变量在哪里被使用；调用方式window.varname

javascript的数据类型：

string number boolean object function symbol

对象类型：

Object Date Array

不包含值的数据类型

null undefined

异常与捕捉

变量提升（hoisting）：函数function声明与变量var声明总是被解释器放到方法体的最顶部；

注意：只有声明会提升，但是初始化不会被提升。

以下两段script一样意思

严格模式：在script中加上“use strict”；

严格模式下，不允许使用未声明的变量，也不允许使用delete删除变量或函数

表单的检查约束：

其中validationMessage是提示的错误信息，validity是是否合法，validity.valid 可等同于checkValidity（）；

call与apply、搭配this使用，可以将this绑定到任何对象上； 在调用函数时第一个参数会成为 this 的值， 即使该参数不是一个对象。

调用形式：对象(方法所属对象).方法.apply/call(新的对象,参数…)；

var let const 的区别： var 不具备块级作用域的特性 ，在{}之外依然能够被访问到

使用var关键字声明的全局作用域变量属于window对象。

使用let关键字声明的全局作用域变量不属于window对象。

使用var关键字声明的变量在任何地方都可以修改。

在相同的作用域或块级作用域中，不能使用let关键字来重置var关键字声明的变量。

在相同的作用域或块级作用域中，不能使用let关键字来重置let关键字声明的变量。

let关键字在不同作用域，或不用块级作用域中是可以重新声明赋值的。

在相同的作用域或块级作用域中，不能使用const关键字来重置var和let关键字声明的变量。

在相同的作用域或块级作用域中，不能使用const关键字来重置const关键字声明的变量

const 关键字在不同作用域，或不同块级作用域中是可以重新声明赋值的:

var关键字定义的变量可以先使用后声明。

let关键字定义的变量需要先声明再使用。

const关键字定义的常量，声明时必须进行初始化，且初始化后不可再修改。

JSON：javascript object notation；

JSON用于存储与传输数据的格式，通常用于服务端向网页传递数据

以键值对的形式存储，不同键值对用逗号隔开，大括号保存对象，方括号保存数组

将json数据转成javascript对象的方法:JSON.parse();

将javascript对象转成json字符串的方法: JSON.stringify();

void关键字：方法不返回任何值，使用语法如下

function

①函数可以赋值给变量

②函数可以自行调用（functtion(){})();

③函数中有一个内置对象，包含了参数的数组，arguments

javascript闭包

一种保护私有变量的机制

可以理解为在外部函数内部声明一个内部函数并调用，且内部函数可以使用外部函数中定义的变量

HTML DOM（Document Object Model）

javascript 可以对页面中所有事件做出反应

各种事件

DOM中创建新的元素

HTMLCollection对象，注意是集合，类似数组，但不是数组

NodeList对象， 文档节点的集合

**javascript prototype:**所有的javascript对象都从原型对象中继承属性与方法

如Date从Date.prototype中继承；Array从Array.prototype中继承

BOM：browser object model

Window对象：所有浏览器的对象，表示浏览器窗口；

多有javascript全局对象，函数及变量都是window对象的成员。

全局变量是window对象的属性；全局函数是window对象的方法
